League Of Legends

Platform that consumes and displays data from JSON file, allowing users to filter, sort and search the characters of the multiplayer videogame according to their roles, skills and attack levels.

Technologies and Features:

  • HTML.
  • CSS.
  • JavaScript.
  • Git.
  • Github.
  • Unit Testing.
  • Figma.
  • Consume of JSON file for data usage.
  • Filter champions according to their skill level.
  • Sort champions according to their skill level.
  • Access to champions statistics.
  • Navigation between pages.
  • Different sections for new players.
  • Responsive design.
