Burger Queen

CRUD and SPA application developed with React, through which a restaurant can take orders using a tablet and send them to the kitchen to be prepared orderly and efficiently, solving the same problem: how to keep the interface and status synchronized.

Technologies and Features:

  • React.
  • HTML.
  • CSS.
  • Git.
  • Github.
  • Bootstrap.
  • Unit Testing.
  • Figma.
  • Consume of JSON Server for data usage.
  • Filter products according to their type (breakfast, meals or drinks).
  • Send orders to kitchen.
  • Access to different section for waiters, administrator and chef.
  • Navigation between pages.
  • Responsive design.
