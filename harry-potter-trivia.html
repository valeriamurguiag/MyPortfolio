Harry Potter Trivia

Platform where users can find Harry Potter trivias, enabling them to answer multiple option questions and displaying their final score using JavaScript logic.

Technologies and Features:

  • HTML.
  • CSS.
  • JavaScript.
  • Git.
  • Github.
  • Figma.
  • Track of user answers to add final score.
  • Two difficulty trivias.
  • Responsive design.
